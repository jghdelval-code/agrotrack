☁️ Sincronización con Google Drive

⚠️ El fichero está abierto como file://
Google Drive no funciona con file://. Sigue el paso 1 de la guía abajo para publicar en GitHub Pages.
☁️
No conectado
Se guarda solo en este dispositivo (localStorage). No se envía a ningún servidor.
📋 PASO 1 — Publicar en GitHub Pages (una vez, gratis, acceso desde cualquier dispositivo)
  1. Ve a github.com y crea una cuenta gratuita si no tienes
  2. Crea un repositorio nuevo llamado agrotrack (público)
  3. Sube el fichero cuaderno-fitosanitario.html al repositorio:
    Clic en Add file → Upload files → arrastra el HTML → Commit changes
  4. Ve a Settings → Pages (menú lateral)
  5. En Source selecciona main y clic Save
  6. En unos segundos tendrás tu URL permanente:
    https://TU_USUARIO.github.io/agrotrack/cuaderno-fitosanitario.html
    Abre esa URL — a partir de ahí siempre la misma, en cualquier dispositivo
✅ Una vez publicado, cuando actualices el fichero HTML solo tienes que volver a subirlo al mismo repositorio y se actualiza automáticamente.
🔑 PASO 2 — Crear credenciales en Google Cloud (una vez)
  1. Ve a console.cloud.google.com
  2. Crea un proyecto nuevo → ponle nombre "AgroTrack"
  3. APIs y servicios → Biblioteca → busca Google Drive API → Habilitar
  4. APIs y servicios → Pantalla de consentimiento OAuth
    → Tipo: Externo → rellena el nombre de la app → guarda
    → En Usuarios de prueba → + Add users → añade tu email de Google
  5. APIs y servicios → Credenciales → + Crear credenciales → ID de cliente OAuth
  6. Tipo de aplicación: Aplicación web
  7. Orígenes autorizados de JavaScript — añade:
    https://TU_USUARIO.github.io 📋 copiar
  8. URIs de redirección autorizados — añade:
    https://TU_USUARIO.github.io/agrotrack/cuaderno-fitosanitario.html 📋 copiar
  9. Clic Crear → se abre un diálogo con tu ID de cliente
    Cópialo y pégalo en el campo de arriba
⚠️ Si ves "Esta app no está verificada" al conectar — es normal para apps personales. Clic en Configuración avanzada → Ir a AgroTrack (no seguro). Solo tú usas esta app.
🔄 PASO 3 — Conectar y sincronizar
  1. Abre la app desde tu URL de GitHub Pages
  2. Pulsa el botón ☁️ Google Drive en la cabecera
  3. Pega tu Client ID en el campo de arriba
  4. Pulsa 🔑 Conectar con Google → elige tu cuenta → acepta permisos
  5. La app carga automáticamente tus datos desde Drive
  6. A partir de ahí, cada cambio se guarda en Drive automáticamente — icono ✅ en la cabecera cuando está sincronizado
📱 En el móvil: abre la misma URL de GitHub Pages en el navegador → conecta Drive → mismos datos que en el ordenador.
